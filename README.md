# doc_auto_handling

Сервис автоматизации обработки **заявок** и **ваучеров** буксира:
приём документов → извлечение полей → сопоставление → расчёт стоимости → БД → экспорт.

Заменяет ручную секретарскую работу: разбор писем-заявок, ввод ваучеров,
сопоставление, расчёт по тарифам с курсом ЦБ РФ и производственным календарём.

## Архитектура (MVP)
Монолит **FastAPI + SQLite (WAL)**, модули пайплайна — чистые функции
(готовы к выносу в отдельный worker при росте нагрузки).

```
app/
  config.py            # настройки + ТАРИФНЫЕ КОНСТАНТЫ (плейсхолдеры — заменить на реальные)
  database.py          # подключение SQLite (WAL), сессии
  models.py            # схема БД (по matching.xls): applications, vouchers, works + справочники
  main.py              # FastAPI: приём, ручной ввод, сопоставление, расчёт, экспорт
  services/
    application_parser.py  # разбор заявок (HTML .eml / текстовый PDF)
    voucher.py             # (заглушка) распознавание ваучеров-сканов — Фаза 3/6
    matching.py            # сопоставление заявка↔ваучер (судно+вид работ+дата)
    calculation.py         # расчёт: тарифы, курс ЦБ, ночь/праздник, сопровождение
    cbr.py                 # курс валют ЦБ РФ (cbr.ru)
    calendar_ru.py         # рабочий/выходной день (isdayoff.ru + откат на выходные)
    export.py              # экспорт Excel/CSV (формат matching.xls), импорт справочников
  web/                 # шаблоны Jinja2 + статика (минимальный UI)
data/                  # SQLite и загруженные файлы (в .gitignore)
```

## Установка и запуск
```bash
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
# открыть http://127.0.0.1:8000
```

## Тарифы
Реальные ставки — коммерческая тайна и в репозитории **не хранятся**.
В `app/config.py` стоят **условные константы** (`RATES`, `RATES_PER_OPERATION`),
чтобы работала логика расчёта. Проставь реальные значения `rate` — логика не меняется.

Правила (подтверждены заказчиком):
- 1 у.е. = 1 USD; ставки заведены только для агента Транс-Агро (90% заявок);
- почасовые услуги тарифицируются **пропорционально минутам**: `ставка × минуты / 60`;
- швартовка/отшвартовка для судов ≥2000 GRT — **за регистровую тонну** (GRT из заявки),
  сумма делится на число буксиров, работавших совместно (строка ваучера «совместно с …»):
  1 буксир → /2, 2 буксира → /3 и т.д.;
- суда <2000 GRT — фиксированная сумма **за операцию** (будни / праздник+ночь);
- ночное время **22:00–06:00**; праздники/выходные — из производственного календаря;
- ледовые условия — ручной флажок в ваучере;
- «Сопровождение» — часы вводятся вручную (авто-разбор примечаний ваучера — позже);
- выручка в рублях = `сумма × курс ЦБ` на дату завершения работ (для рублёвых договоров курс = 1).
